# Claro — AI 광고 제작 과제

> 브랜드: Claro (AI 집중력 & 태스크 관리 앱) | 광고 길이: 약 20초 | 톤앤매너: 클린, 모던, 프리미엄

---

## 과제 개요

멀티모달 AI 도구를 활용해 가상 브랜드 Claro의 20초 광고 영상을 기획·제작하는 과제.  
이미지(Leonardo AI), 영상(Runway ML), 오디오(ElevenLabs + Suno) 각각 다른 AI 도구를 사용하여 씬별로 생성하고 CapCut으로 최종 편집한다.

**핵심 메시지**: Claro — 오늘의 혼돈을, 하나의 흐름으로.  
**슬로건**: Think Clear.

## 광고 구조 (4씬 × 5초)

| 씬 | 구간 | 내용 | 내레이션 |
|----|------|------|---------|
| Scene 1 🔴 기 | 0~5s | 알림 카드들이 사방에서 날아와 화면에 쌓이는 혼돈 | "하루에도 수백 번, 생각이 흩어집니다." |
| Scene 2 🔵 승 | 5~10s | Claro 앱이 탑재된 스마트폰이 천천히 떠오름 | "Claro가 열리는 순간," |
| Scene 3 🟢 전 | 10~15s | 손이 앱을 스크롤하는 집중된 순간 | "오늘 할 일이 선명해집니다." |
| Scene 4 ⚪ 결 | 15~20s | 브랜드명 + 슬로건 + CTA만 남는 미니멀 엔딩 | "Claro. 지금 다운로드하세요!" |

## 사용 도구

| 분류 | 도구 | 용도 |
|------|------|------|
| 이미지 생성 | Leonardo AI (Flux Schnell) | 씬별 키비주얼 생성 |
| 영상 변환 | Runway ML Gen-4 Turbo | Image to Video, 씬별 5초 |
| 내레이션 | ElevenLabs | 한국어 전용 목소리, 4줄 |
| BGM | Suno | 전체 공통 앰비언트 1곡 |
| 최종 편집 | CapCut | 클립 연결 + 오디오 믹스 + 페이드아웃 |

## 프롬프트 이터레이션 요약

총 수정 횟수: 이미지 8회 + 영상 8회 + 오디오 1회

| 씬 | 이미지 수정 | 영상 수정 | 주요 개선 내용 |
|----|:----------:|:--------:|--------------|
| Scene 1 | 5회 | 2회 | 물리적 메모 → 디지털 UI로 방향 전환, 저작권 문제 해결 |
| Scene 2 | 2회 | 4회 | 제품 목업 추가, 폰 뒷면 노출 문제 해결 |
| Scene 3 | 1회 | 0회 | 꺼진 화면 → 앱 켜진 상태로 수정 |
| Scene 4 | 0회 | 3회 | 'o' 튀기기 시도 → 수면 파동 효과로 방향 전환 |
| BGM | — | — | 1회: 음침한 피아노 → 밝은 앰비언트로 전환 |

전체 이터레이션 상세 기록은 `claro-storyboard.md` Section 04 참고.
